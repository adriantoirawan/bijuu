# ⚙️ 1st Gear: EJS Templating Rules (Survival Mode)

*(Scroll to the very bottom of this document to shift gears to another topic)*

EJS causes 90% of syntax errors in Live Codes. Follow these two rules:

1. **`<%= %>`**: Use this to PRINT data to the screen (e.g. `<td><%= movie.name %></td>`).
2. **`<% %>`**: Use this to run SILENT logic (like a `for` loop or `if` statement) without printing anything.
