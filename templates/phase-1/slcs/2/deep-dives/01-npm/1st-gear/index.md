# ⚙️ 1st Gear: What is `npm init -y`? (Survival Mode)

*(Scroll to the very bottom of this document to shift gears to another topic)*

When starting a project, you must run this immediately to create `package.json`. 

- **Why?** It tracks what libraries (Express, EJS) you install so others can run your code.
- **Why `-y`?** It skips the Q&A questionnaire and creates a default file instantly.
