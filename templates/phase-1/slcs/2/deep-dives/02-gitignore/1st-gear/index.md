# вЪЩпЄП 1st Gear: Understanding `.gitignore` (Survival Mode)

*(Scroll to the very bottom of this document to shift gears to another topic)*

You must create a `.gitignore` file and type `node_modules` inside it.

- **Why?** `node_modules` contains tens of thousands of files. If you don't ignore it, pushing to GitHub will take 10 minutes and you might fail your Live Code submission deadline.
